# Python Style Guide (PEP 8 Summary)

1. 변수와 함수명은 **snake_case**, 클래스명은 **PascalCase**로 작성한다.
   예: def calculate_total_price():, class UserProfile:

2. 들여쓰기는 4칸 공백을 사용한다.

3. 줄 길이는 79자 이하를 권장한다.

4. 불필요한 세미콜론(;)과 import 문을 피한다.

5. Docstring은 **""" 세 줄 문자열 """**로 작성하며, 함수의 목적과 파라미터를 명확히 설명한다.

6. 비교 시 `== None` 대신 `is None`을 사용한다.

7. 상수는 모두 대문자와 언더스코어를 사용한다.  
   예: MAX_RETRY_COUNT = 3

8. import 순서는 표준 라이브러리 → 서드파티 → 로컬 모듈 순으로 구분한다.
