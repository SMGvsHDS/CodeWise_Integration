# Java Style Guide (based on Google & Oracle)

1. 클래스명은 **PascalCase**, 메서드명과 변수명은 **camelCase**로 작성한다.
   예: UserService, calculateTotalPrice()

2. 상수(Constant)는 **대문자 + 언더스코어**로 표기한다.
   예: MAX_USER_COUNT, DEFAULT_TIMEOUT

3. 중괄호 `{}`는 **같은 줄에** 배치한다.
   예:
   if (isValid) {
       doSomething();
   } else {
       handleError();
   }

4. 들여쓰기는 4칸 공백을 사용한다.

5. 한 파일에는 **공개 클래스는 하나만** 선언한다.

6. Javadoc을 활용해 메서드의 동작, 매개변수, 반환값을 명확히 기술한다.

7. import 문은 알파벳 순서로 정렬하며 와일드카드(*)는 사용하지 않는다.

8. 메서드는 **하나의 책임만 수행**해야 하며, 이름만으로 기능을 유추할 수 있어야 한다.
