The following Github projects offer examples that you can learn from and possibly even extend:
tut-spring-boot-kotlin
: Sources of
the official Spring + Kotlin tutorial
spring-boot-kotlin-demo
: Regular Spring Boot and Spring Data JPA project
mixit
: Spring Boot, WebFlux, and Reactive Spring Data MongoDB
spring-kotlin-functional
: Standalone WebFlux and functional bean definition DSL
spring-kotlin-fullstack
: WebFlux Kotlin fullstack example with Kotlin2js for frontend instead of JavaScript or TypeScript
spring-petclinic-kotlin
: Kotlin version of the Spring PetClinic Sample Application
